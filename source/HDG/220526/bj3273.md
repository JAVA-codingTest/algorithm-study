## 두 수의 합

양 끝에 포인터를 설정 한 후 풀었습니다

python3 / 41620kb / 132ms

```python
import sys

n = int(sys.stdin.readline())
arr = list(map(int, sys.stdin.readline().split()))
arr.sort()
x = int(sys.stdin.readline())

left, right = 0, n-1
answer = 0

while left < right:
    if arr[left] + arr[right] == x:
        answer +=1
        right-=1
    elif arr[left] + arr[right] > x:
        right-=1
    else:
        left+=1
print(answer)
```
