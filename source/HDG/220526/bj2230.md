## 수 고르기

답안을 참고했습니다

python3 / 34692kb / 224ms

```python
import sys

n, m = map(int, sys.stdin.readline().split())
arr = []
for i in range(n):
    arr.append(int(sys.stdin.readline()))
arr.sort()

left, right = 0,0
answer = sys.maxsize

while left < n and right < n:
    temp = arr[right] - arr[left]
    if temp == m:
        print(m)
        exit()

    if temp<m:
        right+=1
        continue
    left += 1
    answer = min(answer, temp)
print(answer)
```
