## 두 용액

전에 풀었던 문제였지만, 답지를 참고했습니다

python3 / 42312kb / 828ms

```python
import sys

n = int(sys.stdin.readline())
arr = list(map(int, sys.stdin.readline().split()))
arr.sort()
left, right = 0,0
answer = float('inf')

for i in range(n-1):
    start = i+1
    end = len(arr) - 1

    while start <= end:
        mid = (start + end) // 2
        temp = arr[mid] + arr[i]

        if abs(temp) < answer:
            left, right, answer = i, mid, abs(temp)

        if temp > 0:
            end = mid - 1
        else:
            start = mid + 1

print(arr[left], arr[right])
```
